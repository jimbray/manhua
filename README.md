# 简介
H5漫画

浏览地址： http://pinjicake.cn

技术：vue v3.0 + vant + less + axios + qs .
# 环境的准备

推荐使用yarn管理依赖

npm i yarn -g

yarn config set registry htts://registry.npm.taobao.org

安装依赖

yarn 

启动项目

yarn serve 

打包项目

yarn build
