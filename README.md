# 使用方法

1、clone 仓库
> git clone https://github.com/yw738/vue_mh_h5.git
2、安装yarn顺便设置淘宝镜像（可以不设置镜像，如果有yarn跳过此操作）
> npm i yarn -g
> yarn config set registry htts://registry.npm.taobao.org 
3、进入项目目录下载依赖
> yarn
4、运行项目
> yarn serve 
5、打包项目
> yarn build 
6、预览项目
> 双击打开生成在dist目录下的index.html文件即可
